# Programming Tutorial Files
This repo includes tutorials for basic programming concepts for:

 * Java
 * Python 3
 * C/C++
 * Bash
 * Assembly 0x8086

## Topics
 * Variables
 * Conditionals
 * Loops
 * Arrays/Dictionaries
 * Classes
 * I/O
 * Common libraries/packages for each language

## Environment
All the programs were written and compiled on a CentOS 7 vm
